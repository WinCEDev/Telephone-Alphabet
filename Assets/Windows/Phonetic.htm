    About Telephone Alphabet Converter

      This is a simple utility to quickly convert between regular words and their phonetic form. It is useful for reliably relaying information over the phone, such as website addresses, usernames and passwords.

        See Also
        Getting Started

        Getting Started

          When you open the application, the main interface will appear. You can enter text to be converted in the upper text field. You can use the toolbar to quickly copy, paste or undo.

            See Also
            Saving a Text File
            Personalizing the Application

            Saving a Text File

              You can export your typed text and their phonetic form to a text file for later reference or for further processing with other applications. To do this, tap File and then Save As..., or select the disk icon on the toolbar.


                Personalizing the Application

                  You have various options to make the application better fit your needs.

                    Changing the Layout
                    Toggling Auto-Refresh

                    Changing the Layout

                      You can show the input and output fields either horizontally or vertically. To change it, use the layout buttons on the toolbar or alternatively tap View and choose either Arrange Horizontally or Arrange Vertically.


                        See Also
                        Toggling Auto-Refresh

                        Toggling Auto-Refresh

                          By default, the application will convert text while you type, to provide you the results as quickly as possible. However, if you desire to convert longer sentences, you may find that the application will respond slowly. To prevent this, you can turn off auto-update by going to View and setting the Update Automatically option to off.

                          To update the view, tap the Refresh icon on the toolbar, go to View, Refresh, or hit F5 (if available on your device).

                            Setting the Font

                            Setting the Font

                              You can specify an alternate font and size. To do this, select Edit and then Font to configure these options.


                                See Also
                                Zooming In And Out

                                Zooming In And Out

                                  To zoom in and out without changing the stored text size, press Ctrl+Plus and Ctrl+Minus, or alternatively use the menu at View, Zoom, Zoom In and Zoom Out. Here you can also reset the zoom level back to default.
